# T8 — Pattern Screenshot Checklist

Capture screenshots of every Squirrels block pattern at three viewports for documentation, the Treehouse site, and future marketing assets.

**Capture tool:** Chrome DevTools device emulation (or Puppeteer/pageres for automation)  
**Format:** PNG, no browser chrome visible  
**Demo URL:** `https://demo.squirrels.ninja`

---

## Viewports

| Name | Width | Notes |
|------|-------|-------|
| Desktop | 1440px | Full layout |
| Tablet | 768px | Two-column grid check |
| Mobile | 375px | One-column stack, mobile menu |

---

## Setup

Before capturing, create a dedicated "Pattern Preview" page in WordPress:

1. Go to **Pages → Add New**
2. Title: "Pattern Preview — [Pattern Name]"
3. Set Status: **Private** (not publicly indexed)
4. Insert the pattern, save, and screenshot the preview URL

For WooCommerce patterns, use the live demo site with products imported.
