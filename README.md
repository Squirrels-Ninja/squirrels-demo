# squirrels-demo — Demo Site Configuration

> **Status:** Planned · Deployed at [demo.squirrels.ninja](https://demo.squirrels.ninja)

This repository tracks the configuration and content of the Squirrels live demo site.

## What's in here

- `import/` — WXR export of demo content (products, posts, pages, menus)
- `config/` — Theme mod snapshots (customizer settings for each demo)
- `scripts/` — Shell scripts for resetting and re-importing the demo
- `docs/` — Notes on the hosting setup and maintenance procedures

## Demo site

The live demo at [demo.squirrels.ninja](https://demo.squirrels.ninja) runs the Classic Store starter pack with:

- 12 sample WooCommerce products
- Fully configured shop, cart, checkout, and account pages
- Sample blog posts
- Squirrels Customizer set to brand colors

## Maintenance

The demo is reset to a clean state weekly. No real orders or user accounts are processed.
